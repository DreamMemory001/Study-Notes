# 基本算法
## 最大公因数及最小公倍数
* 有两整数a和b：
* ① a%b得余数c
* ② 若c=0，则b即为两数的最大公约数
* ③ 若c≠0，则a=b，b=c，再回去执行①
```c
#include<stdio.h>
int main()
{
	int a,b,c,m,n;
	scanf("%d%d",&a,&b);
	m = a,n = b;
	while(b != 0)
	{
		c = a % b;
		a = b;
		b = c;
	}
	printf("最大公因数%3d\n",a);
	printf("最小公倍数%3d\n",m*n/a);
	return 0; 
}
```
